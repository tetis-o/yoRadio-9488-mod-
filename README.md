Основано на проекте e2002/yoradio (https://github.com/e2002/yoradio). 

# Особенности мода:

1. Основан на версии 0.9.720.
2. Для ESP-32 S3.
3. Для дисплеев ILI9488 и ST7796.
4. Воспроизводит все основные аудио форматы - mp3, aac, flac, vor, opus.
5. Иконки вместо надписей IP, rssi, громкость.
6. Иконка календаря с отображением месяца, числа и дня недели.
7. Иконки для отображения аудио формата в битрейте.
8. Иконки для отображения источника контента - радио или SD карта с номером воспроизводимого трека или канала.
9. Иконки погоды. При отключенной погоде отображаются лого и иконка ёРадио.
10. VU метр с пиками.
11. Изменен шрифт часов.
